# kobra

UCI chess engine

![alt tag](https://github.com/jasper-sinclair/kobra/blob/main/src/kobra.png)

kobra is a work in progress...

it has no search or eval and is limited to random moves but can play a complete game.

- Windows
- 64 bit
- c++17
- bitboards
- hash
- threads
- random
- perft
